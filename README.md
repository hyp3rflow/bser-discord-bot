# bser-discord-bot

영원회귀: 블랙서바이벌 전적을 검색해주는 디스코드 봇입니다.

## Feature

```!전적 (닉네임)``` 입력 시 해당 유저의 정보를 간단하게 요약하여 이미지로 반환합니다.

![example.png](assets/example.png)

## Start
1. ```yarn```으로 패키지를 설치해주세요.
2. .env 파일을 만들고 BOT_TOKEN, API_KEY, API_HOST를 설정해주세요.
3. ```yarn dev```를 이용해 실행해주세요.
